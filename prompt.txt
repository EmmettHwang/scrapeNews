전체적인 흐름은 **"데이터 수집(Backend Worker) -> 저장(DB) -> 서빙(API/Web) -> 알림(Telegram)"** 순서로 구성됩니다.

### 1단계: 개발 환경 설정 및 구조 잡기 (Environment Setup)
가장 먼저 파이썬 3.9 가상환경을 만들고 필요한 라이브러리들을 설치해야 합니다.
* **가상환경 생성:
* **필수 라이브러리 설치:**
    * 웹 프레임워크: `fastapi`, `uvicorn[standard]`
    * 데이터베이스 (ORM): `sqlalchemy` (가장 추천), `alembic` (마이그레이션 도구)
    * 스크래핑: `beautifulsoup4`, `requests` (동적 페이지라면 `playwright`나 `selenium` 고려)
    * 텔레그램: `python-telegram-bot`
    * 스케줄러 (주기적 실행): `apscheduler`
    * 환경변수 관리: .env

### 2단계: 데이터베이스 설계 (Database Modeling)
원격지의 DB를 이용하세요(bitnmeta2.synology.me를 이용하세요)
* **테이블 설계 (News):** - AI 도움 받아서 SQL 문으로 프로그램 내부에 넣어도 됩니다. 
    * `id` (Primary Key)
    * `title` (뉴스 제목)
    * `link` (뉴스 원문 링크 - 중복 수집 방지를 위해 Unique Key로 설정 추천)
    * `summary` (뉴스 요약)
    * `published_at` (기사 작성일)
    * `created_at` (수집된 날짜)

### 3단계: AI 뉴스 스크래핑 모듈 개발 (Scraping Logic)
FastAPI와는 별개로 작동하는 파이썬 함수를 먼저 만듭니다. 이 단계에서는 순수하게 데이터를 가져오는 것에 집중합니다.
1.  **타겟 사이트 선정:** AI 관련 뉴스 사이트 (예: AI 타임스, Google News AI 섹션 등) 선정.
2.  **파싱(Parsing):** HTML 구조를 분석하여 제목, 링크, 본문을 추출합니다.
3.  **테스트:** `print()`를 찍어보며 원하는 데이터가 깔끔하게 나오는지 확인합니다.

### 4단계: 텔레그램 봇 연동 (Telegram Integration)
스크래핑한 데이터를 나에게 보내줄 봇을 만듭니다.
1.  **봇 생성:** 텔레그램에서 `@BotFather`를 통해 봇을 만들고 **API Token**을 발급받습니다.
2.  **Chat ID 확인:** 봇에게 말을 걸고 내 채팅방의 ID를 알아냅니다.
3.  **메시지 전송 함수 작성:** 파이썬 코드로 "Hello World"를 내 텔레그램으로 보내는 테스트를 완료합니다.

### 5단계: 로직 통합 (Integration & Scheduling)
이제 위에서 만든 기능들을 하나로 묶습니다. **APScheduler**가 핵심 역할을 합니다.
1.  **로직 흐름:** 스크래핑 실행 -> DB에 없는 새로운 뉴스인지 확인 -> 없으면 DB 저장 -> **저장된 건에 한해 텔레그램 전송**.
2.  **스케줄링:** "매일 오전 9시" 혹은 "1시간 마다" 실행되도록 스케줄러를 설정합니다. (FastAPI 앱이 실행될 때 스케줄러도 같이 돌도록 설정)

### 6단계: FastAPI 웹 서버 및 UI 구현 (Web App)
수집된 데이터를 웹페이지에 보여주는 단계입니다.
1.  **API 엔드포인트:** `GET /news` 요청 시 DB에서 뉴스 리스트를 JSON으로 반환하는 API를 만듭니다.
2.  **UI (템플릿):** FastAPI는 **Jinja2** 템플릿 엔진을 기본으로 지원합니다.
    * `templates/index.html` 파일을 만듭니다.
    * HTML/CSS로 간단한 뉴스 목록 디자인을 잡고, 파이썬에서 넘겨준 뉴스 데이터를 반복문(`{% for news in news_list %}`)으로 화면에 뿌려줍니다.

### 7단계: 배포 및 운영 (Deployment)
* **Docker:** `Dockerfile`을 작성하여 어디서든 실행 가능하게 만듭니다.
* **서버:** AWS 프리티어나 개인 PC, 혹은 Cloudtype 같은 PaaS 서비스에 배포합니다.
