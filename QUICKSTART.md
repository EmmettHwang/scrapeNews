# ⚡ Railway 배포 - 5분 Quick Start

## 🎯 목표
**5분 안에 Railway에 배포하고 웹사이트 실행하기!**

---

## 📝 준비물 (꼭 확인하세요!)

아래 4개 환경변수 값을 미리 복사해두세요:

```
1️⃣ GOOGLE_API_KEY=___________________
2️⃣ TELEGRAM_TOKEN=___________________
3️⃣ TELEGRAM_CHAT_ID=_________________
4️⃣ DB_PASSWORD=______________________
```

---

## 🚀 배포 시작! (3단계만 하면 끝!)

### 1단계: Railway 로그인 (30초)

1. 브라우저에서 열기: **[https://railway.app](https://railway.app)**
2. 우측 상단 **"Login"** → **"Login with GitHub"** 클릭
3. GitHub 계정으로 로그인

✅ 완료!

---

### 2단계: 프로젝트 배포 (1분)

1. **"New Project"** 클릭
2. **"Deploy from GitHub repo"** 선택
3. **`EmmettHwang/scrapeNews`** 저장소 선택

> 💡 저장소가 안 보이면?
> → **"Configure GitHub App"** 클릭 → `scrapeNews` 체크 → 저장

✅ 자동으로 배포 시작됨! (빌드 로그가 보입니다)

---

### 3단계: 환경변수 설정 (2분) ⚠️ 가장 중요!

1. 왼쪽 사이드바에서 프로젝트 이름 클릭
2. 상단 탭 **"Variables"** 클릭
3. 아래 변수들을 하나씩 추가:

**4번 반복하세요:**

```
New Variable 클릭 → 이름 입력 → 값 입력 → Add 클릭
```

| Variable Name | Variable Value |
|---------------|---------------|
| `GOOGLE_API_KEY` | 실제 구글 API 키 |
| `TELEGRAM_TOKEN` | 실제 텔레그램 봇 토큰 |
| `TELEGRAM_CHAT_ID` | 실제 채팅 ID |
| `DB_PASSWORD` | 실제 DB 비밀번호 |

✅ 환경변수 추가하면 자동으로 재배포됨!

---

### 4단계: 도메인 생성 및 접속 (30초)

1. 상단 탭 **"Settings"** 클릭
2. 아래로 스크롤 → **"Networking"** 섹션 찾기
3. **"Generate Domain"** 버튼 클릭
4. 생성된 URL 클릭!

✅ **배포 완료!** 🎉

---

## 🎊 축하합니다!

이제 생성된 URL로 접속하면 당신의 AI 뉴스 스크래핑 웹앱을 볼 수 있습니다!

**예시 URL**: `https://scrapnews-production.up.railway.app`

---

## 🧪 테스트하기

1. **메인 페이지 확인**: 뉴스 목록이 보이나요?
2. **"뉴스 분석 시작" 버튼 클릭**: 새로운 뉴스 수집 테스트
3. **텔레그램 확인**: 알림이 왔나요?

---

## ❓ 문제가 생겼나요?

### 🔴 웹사이트가 안 열려요!
→ **"Deployments"** 탭에서 로그 확인
→ 환경변수가 모두 입력되었는지 확인
→ 1-2분 기다린 후 다시 시도

### 🔴 데이터베이스 연결 에러!
→ `DB_PASSWORD`가 정확한지 확인
→ MySQL 서버가 외부 접속을 허용하는지 확인

### 🔴 환경변수가 적용이 안 돼요!
→ 변수 이름 대소문자 확인 (정확히 일치해야 함)
→ 값에 불필요한 공백이나 따옴표 제거
→ **"Deployments"** → **"Redeploy"** 클릭
